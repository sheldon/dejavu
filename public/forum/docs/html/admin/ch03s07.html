3.7. HTML in posts (Module)

3.7.1. What is HTML in posts?

This module allow HTML to be used in posts. This includes allowing special characters (eg. UTF-8) that are HTML encoded. Note: BBCode is a safer alternative to HTML, due to the issues that HTML embedded in posts can cause. It is disabled by default. We recommend you leave it that way unless you know what you are doing ;)
