1.4.2.3. Date formatting

Windows does not fully support the date formatting function strftime(). So if you want to change the date formatting (which is configured using $PHORUM['long_date'] and $PHORUM['short_date'] in the language file {phorum dir}/include/lang/yourlanguage.php), please read the PHP documentation for strftime() and its limitations on the Windows platform at http://www.php.net/strftime
