3.2.3.11. Username Restrictions

3.2.3.11.1. What is Username Restrictions?

This module implements configurable features for enforcing user name restrictions. The module will check the user name at registration time and show an error to the user if the new username does not meet the restrictions. It is disabled by default.

3.2.3.11.2. How do I use Username Restrictions?

Once you have enabled the module, you can configure settings such as username length and character preferences in the module settings page.
