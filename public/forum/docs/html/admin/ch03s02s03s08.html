3.2.3.8. Smileys

3.2.3.8.1. What are Smileys?

Smileys are graphical alternatives to the regularly used emoticons, such as :) and :(. This module is enabled by default. As long as the module is enabled, Phorum will render the supplied strings in the settings section as the image which is provided.

3.2.3.8.2. How do I use Smileys?

In the Settings section of the Smiley module you can change the positioning of smilies on the page, add more smilies and other settings.
