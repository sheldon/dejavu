3.4. BBCode (Module)

3.4.1. What is BBCode?

BBCode (Bulletin Board Code) is a system of tags which allows users to add markup to messages (bold, italic, images, links etc) in a safe way. For example, if text in a post was enclosed like so: [i]text[/i], it would be rendered in italics, [b]text[/b] would be rendered in bold, and so on.
