3.2.3.7. Simple Text Replacement

3.2.3.7.1. What is Simple Text Replacement?

This module allows admins to define text replacement in messages. So for example, if you set the module to replace the word "foobar" with "cake", if a user submits a post with "foobar" in it, it will be replaced with "cake". It is disabled by default.

3.2.3.7.2. How do I use Simple Text Replacement?

Once you have enabled the module, click on Settings. Here, you can view the list of existing strings, or add your own. Simply enter a string to match, and what to replace it with. You can also use PCRE's (Perl Compatible Regular Expressions).
