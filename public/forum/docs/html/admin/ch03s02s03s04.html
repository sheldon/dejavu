3.2.3.4. Event Logging

3.2.3.4.1. What is Event Logging?

This module implements an event logging system, which can be used for logging various events. Other modules can use this module for logging purposes too. This module can be useful for tracking the actions of your users and your moderators, or for tracking development issues. It is disabled by default.

3.2.3.4.2. How do I use Event Logging?

First, go the the Modules section in the admin panel and enable the module. Then, click on Settings to configure what you want to be logged. You can view the logs from the Settings page by clicking on View logged events, or if you are trying to find something specific, filter them by selecting Filter logged events.
