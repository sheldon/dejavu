1.2. Quickstart guide

This is the quickstart guide for installing Phorum, aimed at people that are used to working with websites, PHP and databases. If you are having problems following this guide, then please read Section 1.3, “Detailed installation instructions” instead.

  • Unpack Phorum into a directory below your website's document root;
  • Create a database and a full access database user for that database;
  • Edit include/db/config.php.sample, save it as include/db/config.php;
  • Open {phorum url}/admin.php in your web browser and finish the install.
