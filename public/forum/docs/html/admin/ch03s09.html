3.9. Simple Text Replacement (Module)

3.9.1. What is Simple Text Replacement?

This module allows admins to define text replacement in messages. So for example, if you set the module to replace the word "foobar" with "cake", if a user submits a post with "foobar" in it, it will be replaced with "cake". It is disabled by default.
