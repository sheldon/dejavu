3.5.  Can I post announcements?

For announcements, create a forum in which you will post announcement messages. Setup the permissions in such way that only the users who are allowed to create new announcements can post to this forum.

After this, go to the Modules page in the admin interface. There, you can enable the "Announcements" module and edit its settings to point it at the new announcement forum.
